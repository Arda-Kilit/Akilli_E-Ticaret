# 🧠 Smart E-Commerce System

A **Smart E-Commerce Web Application** with a modern, user-friendly, and dynamic structure.
This project was developed to make the shopping experience faster, smarter, and more interactive.

---

## 🚀 Features

- 🛒 Product listing and detail page
- ❤️ Wishlist system
- 🧺 Cart management
- 👤 User profile panel
- 🎨 Modern and responsive design
- ⚡ Fast and dynamic data management (LocalStorage)
- 🔍 Product search and filtering system (optional, can be added)

---

## 🖥️ Technologies Used

- HTML5

- CSS3

- JavaScript (Vanilla JS)

- LocalStorage API
