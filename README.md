# 🧠 Akıllı E-Ticaret Sistemi

Modern, kullanıcı dostu ve dinamik yapıya sahip bir **Akıllı E-Ticaret Web Uygulaması**.
Bu proje, alışveriş deneyimini daha hızlı, akıllı ve etkileşimli hale getirmek amacıyla geliştirilmiştir.

---

## 🚀 Özellikler

- 🛒 Ürün listeleme ve detay sayfası
- ❤️ İstek listesi (Wishlist) sistemi
- 🧺 Sepet (Cart) yönetimi
- 👤 Kullanıcı profil paneli
- 🎨 Modern ve responsive tasarım
- ⚡ Hızlı ve dinamik veri yönetimi (LocalStorage)
- 🔍 Ürün arama ve filtreleme sistemi _(opsiyonel eklenebilir)_

---

## 🖥️ Kullanılan Teknolojiler

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage API
